## 数据库表结构

需要创建以下数据库表：

```sql
-- 待办标签表
CREATE TABLE sys_todo_tag (
    tag_id BIGINT PRIMARY KEY,
    user_id BIGINT,
    tag_name VARCHAR(100),
    tag_color VARCHAR(20),
    sort INT,
    create_by BIGINT,
    create_time DATETIME,
    update_by BIGINT,
    update_time DATETIME,
    deleted INT DEFAULT 0
);

-- 待办与标签关联表
CREATE TABLE sys_todo_tag_relation (
    relation_id BIGINT PRIMARY KEY,
    todo_id BIGINT,
    tag_id BIGINT,
    create_time DATETIME
);

-- 任务评论表
CREATE TABLE sys_task_comment (
    comment_id BIGINT PRIMARY KEY,
    todo_id BIGINT,
    parent_id BIGINT,
    user_id BIGINT,
    user_name VARCHAR(100),
    comment_content TEXT,
    like_count INT DEFAULT 0,
    create_time DATETIME,
    update_time DATETIME,
    deleted INT DEFAULT 0
);

-- 任务附件表
CREATE TABLE sys_task_attachment (
    attachment_id BIGINT PRIMARY KEY,
    todo_id BIGINT,
    attachment_name VARCHAR(255),
    attachment_path VARCHAR(500),
    attachment_type VARCHAR(100),
    file_size BIGINT,
    upload_user_id BIGINT,
    upload_user_name VARCHAR(100),
    download_count INT DEFAULT 0,
    create_time DATETIME,
    deleted INT DEFAULT 0
);

-- 任务回收站表
CREATE TABLE sys_todo_recycle_bin (
    recycle_id BIGINT PRIMARY KEY,
    todo_id BIGINT,
    user_id BIGINT,
    todo_title VARCHAR(255),
    todo_content TEXT,
    delete_by BIGINT,
    delete_by_name VARCHAR(100),
    delete_time DATETIME,
    recover_time DATETIME,
    is_recover INT DEFAULT 0
);
```

## API 接口列表

### 标签管理
- `GET /system/todo/tag/list` - 获取标签列表
- `GET /system/todo/tag/{id}` - 获取标签详情
- `POST /system/todo/tag` - 新增标签
- `PUT /system/todo/tag` - 修改标签
- `DELETE /system/todo/tag/{id}` - 删除标签
- `POST /system/todo/tag/todo/{todoId}` - 为待办添加标签
- `DELETE /system/todo/tag/todo/{todoId}` - 移除待办标签
- `GET /system/todo/tag/todo/{todoId}` - 获取待办的标签

### 评论管理
- `GET /system/todo/comment/list?todoId=` - 获取评论列表
- `GET /system/todo/comment/count?todoId=` - 获取评论数量
- `POST /system/todo/comment` - 添加评论
- `PUT /system/todo/comment` - 修改评论
- `DELETE /system/todo/comment/{id}` - 删除评论

### 附件管理
- `GET /system/todo/attachment/list?todoId=` - 获取附件列表
- `GET /system/todo/attachment/count?todoId=` - 获取附件数量
- `POST /system/todo/attachment/upload?todoId=&file=` - 上传附件
- `GET /system/todo/attachment/download/{id}` - 下载附件
- `DELETE /system/todo/attachment/{id}` - 删除附件

### 回收站管理
- `GET /system/todo/recycle-bin/list` - 获取回收站列表
- `POST /system/todo/recycle-bin/move/{todoId}` - 移动到回收站
- `POST /system/todo/recycle-bin/recover/{recycleId}` - 恢复待办
- `DELETE /system/todo/recycle-bin/{recycleId}` - 永久删除
- `DELETE /system/todo/recycle-bin/clear` - 清空回收站

### 通知管理
- `POST /system/notification/websocket/test` - 发送测试 WebSocket 通知
- `POST /system/notification/email/test?email=` - 发送测试邮件
- `POST /system/notification/todo/remind?todoId=` - 手动触发待办提醒
- `GET /system/notification/config` - 获取通知配置

### 工作流统计
- `GET /system/workflow/stats/overview` - 获取统计概览
- `GET /system/workflow/stats/trend` - 获取趋势统计
- `GET /system/workflow/stats/process-definition` - 获取流程定义统计
- `GET /system/workflow/stats/duration?processInstanceId=` - 获取耗时统计
- `GET /system/workflow/stats/completion` - 获取完成情况统计

### 工作流任务搜索增强
- `GET /system/workflow/task/todo` - 获取待办任务（支持筛选）
  - 参数：`processName`, `startTime`, `endTime`, `taskName`
- `GET /system/workflow/task/done` - 获取已办任务（支持筛选）
  - 参数：`processName`, `startTime`, `endTime`, `taskName`
