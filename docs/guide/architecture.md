# 架构设计

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                         用户层                               │
│                    (Browser / Mobile)                        │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│                        网关层                                │
│                    mp-gateway (8080)                         │
│              路由转发 / 鉴权 / 限流 / 日志                      │
└─────────────────────────────────────────────────────────────┘
                            │
            ┌───────────────┼───────────────┐
            ▼               ▼               ▼
┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐
│   mp-auth       │ │   mp-system     │ │  mp-generator   │
│   (8081)        │ │   (8082)        │ │   (8083)        │
│   认证服务       │ │   系统服务       │ │   代码生成       │
└─────────────────┘ └─────────────────┘ └─────────────────┘
                            │
            ┌───────────────┼───────────────┐
            ▼               ▼               ▼
┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐
│     Nacos       │ │      Redis      │ │      MySQL      │
│   服务发现       │ │     缓存        │ │     数据库       │
└─────────────────┘ └─────────────────┘ └─────────────────┘
```

## 技术架构

### 后端架构

- **Spring Boot 3** - 应用框架
- **Spring Security** - 安全认证
- **MyBatis-Plus** - 数据访问
- **Flowable** - 工作流引擎
- **Hibernate Validator** - 参数校验

### 前端架构

- **Vue 3 Composition API** - 响应式编程
- **Element Plus** - UI 组件
- **Vue Router** - 路由管理
- **Pinia** - 状态管理
- **Axios** - HTTP 客户端

## 目录结构

```
├── mp-auth/             # 认证服务
├── mp-system/           # 系统服务

```
