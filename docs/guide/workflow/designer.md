# 流程设计器

基于 bpmn.js 的在线流程设计器。

## 功能特性

### 1. 画布操作

- 拖拽式节点添加
- 连线创建
- 节点删除
- 撤销/重做
- 缩放/适应屏幕

### 2. 节点类型

| 节点类型 | 说明 |
|---------|------|
| Start Event | 开始事件 |
| End Event | 结束事件 |
| User Task | 用户任务 |
| Service Task | 服务任务 |
| Exclusive Gateway | 排他网关 |
| Parallel Gateway | 并行网关 |
| Sequence Flow | 顺序流 |

### 3. 属性配置

选中节点后，在右侧属性面板配置：

**用户任务属性：**
- 节点 ID
- 节点名称
- 处理人类型（指定用户/部门负责人/岗位/变量）
- 处理人
- 办理表单

**服务任务属性：**
- 服务类
- 方法
- 结果变量

**条件分支属性：**
- 条件表达式

### 4. 表单配置

点击「关联表单」按钮，配置：

- 启动表单：流程发起时填写
- 办理表单：任务处理时填写

### 5. 导入导出

- 保存流程定义
- 导出 BPMN XML
- 导出 SVG 图片
- 导出 PNG 图片
- 导出 PDF

### 6. 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Delete | 删除选中 |
| Ctrl+F | 适应屏幕 |
